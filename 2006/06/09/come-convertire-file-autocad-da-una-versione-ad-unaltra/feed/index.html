]]> Di: Andrea Borruso http://blog.spaziogis.it/2006/06/09/come-convertire-file-autocad-da-una-versione-ad-unaltra/comment-page-1/#comment-3530 Andrea Borruso Tue, 25 Mar 2008 21:08:49 +0000 http://blog.spaziogis.it/2006/06/09/come-convertire-file-autocad-da-una-versione-ad-unaltra/#comment-3530 Ciao Elvin,
il .pdf non è un formato che conserva tutte le informazioni dei layer da cui è stato generaro. Normalmente prepari un .pdf o per la stampa, o per la visualizzazione a video.
Se parti da un file vettoriale in formato .shp e poi esporti da questo un file .pdf, perderai quasi certamente sia le informazioni sulle coordinate del file GIS che la tipica tabella con le informazioni correlate.
Se il file di input è un raster può rimanere qualcosa in più.
Se il tuo pdf mantiene alcune delle informazioni vettoriali puoi provare ad aprirlo e modificarne le geometrie con Adobe Illustrator (non mi viene in mente alcun programma free), esportarlo in formato dxf ed importare questo nel tuo software GIS; dovrai sicuramente però georeferenziare questo file.
Esiste un formato .pdf specializzato che può contenere al suo interno le classiche informazioni di un file spaziale – il GeoPDF – ma non credo sia il tuo caso.

le soluzioni sono molteplici, ma tu non hai mi hai dato molti dettagli.
Te ne propongo una che ti consente di creare un pdf anche senza avere autocad:
- scarica e installa pdfcreator, un programma opensource che installa una stampante virtuale, che stampa (crea) file .pdf
- scarica e installa DWG Trueview dell’Autodesk, con il quale potrai visualizzare il tuo file dwg (se hai autocad non installarlo)
- stampa il file da DWG Trueview (o da autocad) usando la stampante virtuale creata dall’installazione di pdfcreator
- la stampa del file produrrà un file .pdf

Fammi sapere
