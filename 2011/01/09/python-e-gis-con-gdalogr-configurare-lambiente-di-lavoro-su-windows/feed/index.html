Ad es. nelle varie distribuzioni di Ubuntu basta digitare da shell:

sudo apt-get install python-software-properties
sudo add-apt-repository ppa:ubuntugis/ubuntugis-unstable
sudo apt-get install gdal-bin python-gdal

NB: python 2.6 è già presente di default su Ubuntu

A proposito della nota finale, invece, con il fresco rilascio di GDAL 1.8 sono finalmente disponibili gli installer per win che diceva Andrea. Li potete trovare qui secondo diversi compilatori ed architetture. Happy coding!
