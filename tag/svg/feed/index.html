Le novità più immediate sono legate al rendering vettoriale in formato SVG in Mozilla/Firefox e Opera, e in VML in Internet Explorer. Ma ecco la rassegna ufficiale delle nuove features di questa versione:

  • In-browser vector support using SVG/VML
  • Reworked controls to better support extensibility
  • Added support for toolbar style control deployment
  • Substantial enhancements to existing Layer classes, particularly WFS
  • A massive effort to track down and eliminate memory leaks of JavaScript objects and DOM node
  • Tons and tons of bug fixes – a total of 183 tickets closed, more than any other release.
